nalgebra is a linear algebra library written using the [Rust programming language](https://www.rust-lang.org/).
It targets applications requiring efficient vector and matrix operations, as well as geometric transformations.
It is designed to be fast and comprehensive.

nalgebra is free and open-source, released under the Apache 2.0 license. It is developed by the [Dimforge](https://dimforge.com)
open-source company. You can support us by sponsoring us on [GitHub sponsor](https://github.com/sponsors/dimforge).
